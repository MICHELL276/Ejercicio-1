GIT Y GITHAB

Concepto de Git

Git es un sistema de control de versiones distribuido. esto significa que cada desarrollador tiene una copia completa del historial del proyecto en su equipo local y no depende de un servidor para trabajar.

Concepto de GitHub

GitHub es una web que permite almacenar y gestionar proyectos que utilizan Git como sistema de control de versiones. fue lanzada en 2008 y fue adquirida por Microsoft en 2018. Es la plataforma mas popular en el mundo para alojar codigo, colaborar en proyectos de software y gestionar versiones de manera remota.
